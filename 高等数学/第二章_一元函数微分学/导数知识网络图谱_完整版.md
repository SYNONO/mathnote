### 3.2 通向积分的桥梁

```
导数 → 积分 发展路径

1. 原函数概念
   F'(x) = f(x) ⟹ F是f的原函数

2. 不定积分
   ∫f(x)dx = F(x) + C

3. 定积分（牛顿-莱布尼茨公式）
   ∫[a,b] f(x)dx = F(b) - F(a)

4. 微积分基本定理
   d/dx ∫[a,x] f(t)dt = f(x)
```

## 七、综合连接示例

### 7.1 一题多解展示连接

**例题**：求 f(x) = x²sin(1/x) (x≠0), f(0)=0 在x=0处的导数

```
方法连接网：
1. 定义法 → 极限计算 → 夹逼定理
2. 连续性 → 可导性判断 → 导数存在性
3. 振荡函数 → 有界性 → 极限为0
```

### 7.2 知识综合应用

**综合题**：研究函数 f(x) = x³ - 3x + 1

```
知识调用链：
导数 → f'(x) = 3x² - 3
     ↓
单调性 → f'(x) = 0 得 x = ±1
     ↓
极值 → f(-1) = 3, f(1) = -1
     ↓
二阶导数 → f''(x) = 6x
     ↓
凹凸性 → x<0凸，x>0凹
     ↓
拐点 → (0, 1)
     ↓
函数图像 → 完整性态
```

## 九、核心要点总结

### 导数在数学体系中的地位

1. **承上启下**：连接极限与积分

- 导数不是孤立的概念，而是数学网络的核心节点

- 深入理解导数能打通整个微积分体系
