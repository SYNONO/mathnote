## 📖 定义

有些函数，对其定义域内自变量不同的取值，其对应法则不能用一个统一的数学表达式表示，而要用两个或两个以上的数学式子表示，这类函数称为**分段函数**。

> ⚠️ **核心理解**：分段函数虽然用多个解析式表示，但它是**一个函数**，而不是多个函数。就像一条路虽然有不同路段（高速、普通公路），但仍是一条完整的路。

## 📐 分段函数的标准表示

### 基本格式
分段函数 f(x) 的标准表示：

```
        ┌ f₁(x),  当 x ∈ D₁
        │ f₂(x),  当 x ∈ D₂
f(x) =  ┤  ⋮         ⋮
        └ fₙ(x),  当 x ∈ Dₙ
```

其中 D₁, D₂, ..., Dₙ 是定义域的互不相交的子集。

### 🎯 例1：基础分段函数
```
        ┌ sin x,    当 x < 0
f(x) =  ┤
        └ x² + 1,   当 x ≥ 0
```

**定义域**：(-∞, +∞)

**图形理解**：
- x < 0 时，是正弦曲线
- x ≥ 0 时，是抛物线向上平移1个单位
- 在 x = 0 处，左极限是 sin(0⁻) = 0，函数值是 f(0) = 1

## 📊 常见分段函数类型详解

### 1. 绝对值函数

#### 基本绝对值函数
|x| 的分段表示：
```
        ┌ -x,  当 x < 0
|x| =   ┤
        └  x,  当 x ≥ 0
```

#### 🎯 例2：含绝对值的复合函数
函数 f(x) = 2 - |1 - x²| 的化简：

**步骤1**：分析 1 - x² 的正负性
- 当 1 - x² ≥ 0 时，即 -1 ≤ x ≤ 1
- 当 1 - x² < 0 时，即 x < -1 或 x > 1

**步骤2**：去绝对值
```
            ┌ 2 - (1 - x²) = 1 + x²,   当 -1 ≤ x ≤ 1
f(x) =      ┤
            └ 2 - (x² - 1) = 3 - x²,   当 x < -1 或 x > 1
```

### 2. 取最值函数

#### 🎯 例3：max{x, x²} 的分段表示

**分析过程**：
1. 令 x = x²，解得 x = 0 或 x = 1
2. 比较各区间内 x 和 x² 的大小：
   - x < 0：x < 0 < x²，取 x²
   - 0 ≤ x ≤ 1：x² ≤ x，取 x
   - x > 1：x < x²，取 x²

**结果**：
```
                ┌ x²,  当 x < 0
max{x, x²} =    ┤ x,   当 0 ≤ x ≤ 1
                └ x²,  当 x > 1
```

### 3. 符号函数（Sign Function）

```
            ┌ -1,  当 x < 0
sgn(x) =    ┤  0,  当 x = 0
            └  1,  当 x > 0
```

**几何意义**：表示 x 的符号（正、负、零）

### 4. 取整函数（Floor Function）

**定义**：[x] 表示不超过 x 的最大整数

**例子**：
- [3.7] = 3
- [-2.3] = -3（注意不是 -2）
- [5] = 5

**分段表示**（部分）：
```
        ┌  ⋮
        │ -2,  当 -2 ≤ x < -1
[x] =   ┤ -1,  当 -1 ≤ x < 0
        │  0,  当  0 ≤ x < 1
        │  1,  当  1 ≤ x < 2
        └  ⋮
```

## 💡 分段函数的处理技巧

### 技巧1：化简含绝对值的函数

**核心步骤**：
1. 找出使绝对值内部为零的点（分界点）
2. 判断各区间内绝对值内部的正负性
3. 去掉绝对值，写出分段表达式

### 技巧2：复合分段函数

对于 f[g(x)] 型的复合：
1. 先求出 g(x) 的值域
2. 根据 g(x) 的值判断使用 f 的哪段表达式

### 🎯 例4：复合分段函数（考研真题类型）

设 f(x) = {sin x/x, 当|x|≤1; 0, 当|x|>1}，求 f[f(x)]。

**解答**：
1. 分析 f(x) 的值域：
   - 当 |x| ≤ 1 时，|sin x/x| ≤ 1
   - 当 |x| > 1 时，f(x) = 0

2. 因此 |f(x)| ≤ 1 恒成立

3. 所以 f[f(x)] = sin[f(x)]/f(x)
   - 当 f(x) ≠ 0 时，f[f(x)] = sin[f(x)]/f(x)
   - 当 f(x) = 0 时，根据极限定义，f[f(x)] = 1

**结论**：f[f(x)] = 1（处处为1）

## 📊 分段函数的图像理解

### 绘图要点
1. **分段绘制**：在每个区间内单独绘制对应的函数图像
2. **端点处理**：
   - 实心点（●）：该点属于函数
   - 空心点（○）：该点不属于函数
3. **连续性判断**：检查分段点处的左右极限

### 🎯 例5：绘制分段函数图像

```
        ┌ x + 1,    当 x < 0
f(x) =  ┤ x²,       当 0 ≤ x < 2
        └ 4,        当 x ≥ 2
```

**绘图分析**：
- x = 0 处：左极限 = 1，函数值 = 0（跳跃间断）
- x = 2 处：左极限 = 4，函数值 = 4（连续）

## 🎯 易错警示

### ❌ 错误1：混淆分段点的归属
```
错误写法：f(x) = {x², x < 0; x², x > 0}  // x=0 无定义！
正确写法：f(x) = {x², x < 0; x², x ≥ 0}
```

### ❌ 错误2：绝对值化简错误
对于 |x - 1|：
```
错误：|x - 1| = {x - 1, x > 0; -(x - 1), x < 0}
正确：|x - 1| = {x - 1, x ≥ 1; -(x - 1), x < 1}
```

### ❌ 错误3：取整函数理解错误
```
错误：[-2.3] = -2
正确：[-2.3] = -3  // 不超过-2.3的最大整数是-3
```

## 🔑 解题策略总结

1. **看到绝对值**→ 找零点，分区间讨论
2. **看到max/min**→ 令函数相等，找交点，比大小
3. **看到复合**→ 先求内层值域，再代入外层
4. **看到取整**→ 画数轴，标整数点
