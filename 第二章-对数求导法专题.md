# 第二章 - 对数求导法专题

## 一、核心概念

### 什么是对数求导法？
对数求导法是通过**先取对数，后求导**的方法，特别适用于：
- 幂指函数 y = f(x)^g(x)
- 多个因式的乘除
- 复杂根式表达式

### 核心公式
对于幂指函数 $y = f(x)^{g(x)}$：
$$y' = f(x)^{g(x)} \cdot [g'(x) \cdot \ln f(x) + g(x) \cdot \frac{f'(x)}{f(x)}]$$

## 三、三大经典类型

### 类型1：幂指函数 ⭐最重要

#### 基本形式：$y = f(x)^{g(x)}$

**示例1**：$y = x^x$
```
解法：
ln y = x·ln x
y'/y = ln x + x·(1/x) = ln x + 1
y' = x^x(ln x + 1)
```

**示例2**：$y = x^{\sin x}$
```
解法：
ln y = sin x · ln x
y'/y = cos x · ln x + sin x/x
y' = x^(sin x) · (cos x · ln x + sin x/x)
```

**示例3**：$y = (x^2+1)^{x^3}$
```
解法：
ln y = x³ · ln(x²+1)
y'/y = 3x² · ln(x²+1) + x³ · 2x/(x²+1)
y' = (x²+1)^(x³) · [3x²·ln(x²+1) + 2x⁴/(x²+1)]
```

### 类型2：多因子乘除

#### 基本形式：$y = \frac{f_1(x) \cdot f_2(x) \cdots}{g_1(x) \cdot g_2(x) \cdots}$

**示例**：$y = \frac{(x-1)(x-2)(x-3)}{(x+1)(x+2)}$
```
解法：
ln y = ln|x-1| + ln|x-2| + ln|x-3| - ln|x+1| - ln|x+2|
y'/y = 1/(x-1) + 1/(x-2) + 1/(x-3) - 1/(x+1) - 1/(x+2)
y' = y · [上述结果]
```

### 类型3：复杂根式

#### 基本形式：含有高次根号或分数指数

**示例**：$y = (x-1)^2 \sqrt[3]{\frac{(x+1)(x^2-3)}{(1+4x)^2}}$
```
解法：
ln y = 2ln|x-1| + (1/3)ln|x+1| + (1/3)ln|x²-3| - (2/3)ln|1+4x|
y'/y = 2/(x-1) + 1/[3(x+1)] + 2x/[3(x²-3)] - 8/[3(1+4x)]
y' = y · [上述结果]
```

## 四、解题步骤（口诀：取对-展开-求导-乘回）

### Step 1：取对数
两边同时取自然对数：$\ln y = \ln[f(x)]$

### Step 2：展开化简
利用对数运算法则：
- $\ln(AB) = \ln A + \ln B$
- $\ln(A/B) = \ln A - \ln B$
- $\ln(A^n) = n\ln A$

### Step 3：两边求导
注意：$\frac{d}{dx}(\ln y) = \frac{1}{y} \cdot \frac{dy}{dx}$

### Step 4：乘回原函数
$\frac{dy}{dx} = y \cdot [\text{求导结果}]$

## 五、易错点警示 ⚠️

### 1. 幂指函数陷阱
❌ 错误：把 $x^x$ 当作 $x^n$ 处理
✅ 正确：指数也是变量，必须用对数求导

### 2. 符号处理
- 取对数时要加绝对值：$\ln|x-1|$
- 但最终结果中通常省略绝对值符号

### 3. 链式法则遗忘
- $(x^2-3)' = 2x$，不要忘记内函数求导
- $(1+4x)' = 4$，系数不能丢

### 4. 最后一步
- 别忘记乘回原函数 $y$
- 可以不化简，保持 $y \cdot [\cdots]$ 的形式

## 六、记忆技巧 💡

### 1. 幂指函数口诀
"底变指不变，指变底对数"
- 对底求导：正常求导
- 对指求导：乘以 $\ln(\text{底})$

### 2. 判断使用时机
"三个一"原则：
- 一看是否幂指
- 二数因子个数
- 三查根号次数

### 3. 根号处理
根号次数在分母：
- $\sqrt{x} = x^{1/2}$ → 对数后系数 $\frac{1}{2}$
- $\sqrt[3]{x} = x^{1/3}$ → 对数后系数 $\frac{1}{3}$
- $\sqrt[n]{x} = x^{1/n}$ → 对数后系数 $\frac{1}{n}$

## 七、对比表格

| 方法 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| 普通求导 | 简单函数 | 直接 | 复杂时易错 |
| 对数求导 | 幂指、多因子 | 化简计算 | 需要额外步骤 |
| 公式法 | 幂指函数 | 一步到位 | 公式复杂 |
